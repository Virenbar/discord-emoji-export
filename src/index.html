How to get Token

Using network tab

  • Open development console (F12 or Ctrl + Shift + I)
  • Switch channel or server in Discord
  • Go to Network tab
  • Turn on XHR filter
  • Select any request
  • Find authorization header

Using application tab (browser only)

  • Open development console (F12 or Ctrl + Shift + I)
  • Go to Application tab
  • Select local storage
  • Search for token key
